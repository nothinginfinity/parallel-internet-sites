# TrueBuild — Parallel Internet Site Example

This directory contains the demo implementation spec for TrueBuild's Parallel Internet Site.

TrueBuild is the first client demo for the Parallel Internet Sites product. The goal is to demonstrate a measurable increase in AI mentions and citation accuracy for TrueBuild in response to business credit building prompts.
